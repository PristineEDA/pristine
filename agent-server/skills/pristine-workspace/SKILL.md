# Pristine Workspace

Pristine is an Electron, Vite, React, and TypeScript editor application. Keep changes local to the requested feature and follow existing component, IPC, and test patterns.

Use the agent's read and search tools before proposing edits. Propose file updates through pending changes instead of writing directly. Propose shell commands instead of executing them directly.

Prefer small focused changes, strict TypeScript, and tests that exercise the behavior touched by the change.
